### Project description
* Black and white colorization project has the goal of turning black and white photos into properly distributed color images.
* Method is presented in slide and report.
* Source code is pre-produced above.

### Clone the repository
Turn on terminal
```sh
git clone https://github.com/imhwy/BlackandWhiteColorization.git
cd ComputerVision-CS231
```

### Install requirement
Using pip/conda
```sh
pip install requirement.txt
```

### Running project
Change the image path in file main.py
```sh
parser.add_argument("-i", "--img_path", type=str, default="a.jpg")
```
Run code
```sh
python main.py
```
